Selecting a dataset

In order to view or evaluate your previously uploaded data, select a dataset or a dataset task from the side navigation tree.

Creating new dataset

For evaluating new eye tracking data you need to create a dataset first. Then you can proceed to create the so-called 'dataset tasks' which contain eye tracking data recorded from one screen. For a complex datasets consisting of multiple such screens, create several dataset tasks (each containing data from one screen).

Uploading to an existing dataset

If you wish to upload raw eye tracking data to an already existing dataset, please select the particular dataset and click the 'Add task' button under the tasks list. Fill out the form and once the uploading has finished, you are good to go.

Scanpath data

Currently, the only supported input data format is the default Tobii Studio export. In this export, the scanpath data are formatted as a CSV file with tab separator. Required are the following columns: ParticipantName, FixationIndex, GazeEventType, GazeEventDuration, FixationPointX (MCSpx), FixationPointY (MCSpx) and MediaName. However, only the first 4 listed column values are currently being used - remaining columns can contain any data.

This file is later formatted on the server side, but alternatively you can do the formatting on your own and upload an already formatted dataset.

AOI data

Areas of interest data format is not really standardized in the Tobii Studio export, so please refer to this file instead. As you can see in the file, areas of interest are limited to 4 vertices each, so please stick to rectangle and square shapes.

Again, AOI data are later formatted in order to speed up calculations, but alternatively you can upload an already formatted AOI file.
